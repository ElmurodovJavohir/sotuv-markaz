# ADR-0003: Infinity Sales → Sotuv Markaz rebrand

**Sana:** 2026-04-30
**Status:** Qabul qilingan
**Qaror beruvchi:** Javokhir, Malika

## Kontekst

Platforma dastlab "Infinity Sales" nomi bilan ishlab chiqilgan. Bozor (O'zbekiston) uchun o'zbekcha nom kerakligi aniqlandi.

## Qaror

- Yangi nom: **Sotuv Markaz**
- Domen: **sotuv-markaz.uz**
- Barcha joyda o'zgartirish: Notion, codebase, Figma, marketing

## O'zgartirilgan joylar

- [x] Notion — barcha sahifalar va task notlari (2026-04-30)
- [ ] Backend codebase — seed data, SEO meta, company services
- [ ] Frontend codebase — og:title, meta tags, i18n keys
- [ ] CLAUDE.md fayllar (backend, frontend, learning, SHARED.md)
- [ ] Figma fayl nomi
- [ ] Logo (agar o'zgarsa)

## Oqibat

- Codebase da repo nomlari hali `infinitysales-*` — o'zgartirish katta refactor, hozircha qoladi
- Internal kod da `infinity` reference lari muammo qilmaydi — bu faqat user-facing nom
